Privacy Policy

Macro Refiner for Gorgias

Last updated: February 2026

Overview

Macro Refiner for Gorgias is a Chrome extension that helps customer support agents generate AI-powered reply suggestions. We are committed to protecting your privacy.

Data Collection

When you use the extension, the following data is temporarily processed to generate reply suggestions:

  • Customer first name (from Gorgias ticket)
  • Ticket subject and message content
  • Order information (if available)

Data Usage

This data is:

  • Sent to our API server solely for generating AI-powered replies
  • Processed in real-time and not stored after the response is returned
  • Never sold or shared with third parties
  • Never used for advertising or marketing purposes

Data Storage

We do not permanently store any personal data. The extension may use local browser storage only to save user preferences.

Third-Party Services

The extension uses the Anthropic Claude API to generate reply suggestions. Data sent to this service is subject to Anthropic's Privacy Policy.
